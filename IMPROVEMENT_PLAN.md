# X-ray Denoiser 改进任务清单

## 已识别的Bug和问题

### 1. 图像深度处理问题
- [ ] 当前代码只处理8位图像(uint8)，无法处理16位、32位浮点图像
- [ ] 医学X-ray通常是16位深度
- [ ] 需要添加自动深度检测和归一化处理

### 2. 算法稳定性问题
- [ ] 小尺寸图像处理时可能崩溃
- [ ] 某些参数组合会导致除零错误
- [ ] 内存不足时没有优雅降级
- [ ] 异常处理不完善

### 3. 缺少神经网络算法
- [ ] 需要添加基于深度学习的去噪算法
- [ ] 可以实现DnCNN、U-Net等轻量级模型
- [ ] 提供预训练模型下载

### 4. 其他改进
- [ ] 添加批处理功能
- [ ] 添加参数调节滑块
- [ ] 支持更多图像格式（DICOM）
- [ ] 添加进度条真实进度显示

## 测试计划
1. 测试不同深度的图像（8位、16位、32位浮点）
2. 测试极小图像（10x10像素）
3. 测试超大图像（10000x10000像素）
4. 测试各种格式（PNG、JPG、TIFF、DICOM）
5. 测试所有算法稳定性
